Factoral Calculator

A factorial is represened by the Symbol "!". It will mutilply the input number by each proceeding whole number until 1 is reached.

Examples

  • 5! = 5x4x3x2x1 = 120
  • 7! = 7x6x5x4x3x2x1 = 5,040
  • 10! = 10x9x8x7x6x5x4x3x2x1 = 3,628,800

As you can see for small numbers, the factorial of that number can grow very fast!

Try it your self. Enter a whole number between 1-120.
